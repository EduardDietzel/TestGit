Описание занятия: 
1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?

- Создаем новый репзиторий на github.com
- Копируем ключ SSH (ссылку)
- Открываем в git bash директорию, в которой хотим скопировать репозиторий из github.com
- Набираем git clone и ключ (ссылку)
- Далее входим в директорию созданного на компьютере репозитория
- cd <название папки>/
- ls -la


2. Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?
- создаем репозиторий на компьютере (в определенном директории)
- mkdir <name_repo> (например TestGit)
- заходим в эту папку: cd TestGit/
- git init
- далее создаем новый репозиторий на github.com с таким же названием TestGit
- в консоли набираем git remote add origin и ссылку на ключ нового репозитория на гитхабе
- git branch -M main
- git push -u origin main


3. Как получить на локальный компьютер изменения с github.com
- используем команду git pull

4. Какую информацию нам дают команда :
git status (показывает файлы, в которых были сделаны изменения, сотояние гит на данный момент)
git log (показывает историю изменений, коммитов (автор, время, дата))
git remote -v (показывает, из какого места на github.com связан наш репозиторий на локальном компьютере)
